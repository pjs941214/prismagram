# prismagram
Instagram clone with Express + Prisma + React and React Native

## User Stories

- [V] Create account
- [V] Request Secret
- [V] Confirm Secret (Log in)
- [V] Like / Unlike a photo
- [V] Comment on a photo
- [V] Search by user
- [V] Search by location
- [ ] See the feed
- [ ] See user profile
- [ ] Follow / Unfollow user
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit / Delete the photo
